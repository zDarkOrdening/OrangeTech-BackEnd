### Java 

* O Java atualmente pode ser considerada uma linguagem multiparadigma,
porém é notável a sua vocação para estruturas orientadas a objetos.

* Em Java tudo é escrito em termos de classes e objetos.

* Pilares da POO:

	- Encapsulamento
	- Abstração
	- Herança
	- Polimorfismo

* Java funciona em uma plataforma independente,
o código-fonte em java é compilado para bytecode e esse mesmo bytecode,
não está vinculado a nenhuma plataforma.

* Java é portatil, seguindo o conceito de WORA (Write Once Run Anywhere),
ou seja, desenvolvedores podem obter o mesmo resultado do código em qualquer máquina/dispositivo,
a razão por trás disso é a JVM e o bytecode.

* Java é uma linguagem de tipagem forte e estática, além de possuir uma sintaxe case sensitive,
é uma linguagem robusta e segura.

* Java é uma linguagem compilada e interpretada ao mesmo tempo, os programas escritos em Java
são compilados para gerar arquivos em bytecode e a JVM (Java Virutal Machine) interpreta esses
arquivos bytecode durante a execução.

* Java é uma linguagem multi-thread, ou seja executa várias threads simultaneamente, o que gera alta performance de execução.

* A linguagem Java é baseada em C e C++.

* Maven é uma ferramenta de automação de compilação.

* Packages:
	com = comercial
	org = organização / opensource

*Java Beans
